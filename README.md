# Teams Avatar Generator

A small web app to generate team avatars with various fonts (including custom TTF).

## GitHub Pages

1. Create a repository on GitHub (e.g. `IconMaker`).
2. Push the project:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR-USERNAME/IconMaker.git
   git branch -M main
   git push -u origin main
   ```
3. Go to **Settings** → **Pages** → Source: **Deploy from a branch** → Branch: **main** → Folder: **/ (root)** → Save.
4. After a few minutes, the site will be available at:
   **https://YOUR-USERNAME.github.io/IconMaker/**

The `index.html` entry page automatically redirects to the app (`teamsIconPicker.html`).
